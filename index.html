Happy Tree Friends Wiki is a fan-series branching off of Happy Tree Friends created on October 7th, 2023 by RealHappyTreeFriends on YouTube.

Stretching over 50 episodes, This website aims to show you all of them!

The show contains many characters starring in the original show, As well as many new ones.

Started as an inside joke, We only went up from there.
